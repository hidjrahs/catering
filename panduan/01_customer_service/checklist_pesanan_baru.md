# ✅ Checklist — Membuat Pesanan Baru (Customer Service)

Gunakan checklist ini setiap kali menerima dan memasukkan pesanan baru dari pelanggan.

---

## 📞 Sebelum Input ke Sistem

- [ ] Konfirmasi nama lengkap pelanggan
- [ ] Catat nomor HP yang bisa dihubungi
- [ ] Catat alamat pengiriman/lokasi event
- [ ] Konfirmasi tanggal & waktu event
- [ ] Konfirmasi lokasi/venue event
- [ ] Konfirmasi jumlah tamu yang makan
- [ ] Konfirmasi jumlah total undangan
- [ ] Tanyakan jenis event (pernikahan, sunatan, ulang tahun, dll)
- [ ] Tanyakan paket menu yang diinginkan
- [ ] Catat menu-menu yang dipesan beserta estimasi porsi
- [ ] Tanyakan ada tidaknya permintaan khusus (alergi, halal, vegetarian, dll)
- [ ] Tanyakan biaya-biaya tambahan (transport, dekorasi, tenda, dll)
- [ ] Konfirmasi apakah ada uang muka (DP) dan jumlahnya

---

## 💻 Saat Input ke Sistem

- [ ] Login ke sistem (`http://localhost:8008`)
- [ ] Buka menu **Customer Service**
- [ ] Klik tombol **"+ Tambah Pesanan"**

### Data Pelanggan
- [ ] Cari pelanggan di sistem (ketik nama/HP)
  - [ ] *(Pelanggan lama)* Pilih dari saran yang muncul
  - [ ] *(Pelanggan baru)* Ketik nama → isi HP dan alamat

### Data Event
- [ ] Isi **Tanggal Event**
- [ ] Isi **Waktu Mulai Event**
- [ ] Isi **Tanggal & Jam Antar**
- [ ] Isi **Venue/Lokasi**
- [ ] Isi **Jumlah Tamu**
- [ ] Isi **Jumlah Undangan**
- [ ] Pilih **Jenis Event**
- [ ] Pilih **Paket Menu**

### Menu Pesanan
- [ ] Tambahkan semua menu yang dipesan
- [ ] Verifikasi harga per menu sudah benar
- [ ] Isi catatan khusus per menu (jika ada)

### Biaya Tambahan
- [ ] Tambahkan semua rincian biaya tambahan
- [ ] Pastikan nama, qty, dan harga sudah benar

### Catatan & DP
- [ ] Isi catatan umum pesanan
- [ ] Isi catatan khusus/permintaan spesial
- [ ] Isi jumlah uang muka (DP) jika ada

---

## 📄 Setelah Menyimpan

- [ ] Sistem berhasil menyimpan (muncul notifikasi hijau)
- [ ] Nomor tiket pesanan sudah terbuat
- [ ] Download/cetak **PDF Laporan Pesanan**
- [ ] Berikan salinan PDF ke pelanggan (WhatsApp/email/cetak)
- [ ] Catat nomor tiket pesanan untuk referensi komunikasi dengan pelanggan
- [ ] Informasikan ke pelanggan bahwa pesanan sedang **menunggu verifikasi**

---

## 🔄 Pemantauan Lanjutan

- [ ] Pantau status pesanan berubah dari `pending` → `approved` (oleh Cost Controlling)
- [ ] Jika ada perubahan dari pelanggan, segera update sebelum status `approved`
- [ ] Konfirmasi ke pelanggan setelah status menjadi `approved`

---

> 📌 **Simpan checklist ini** di meja kerja Anda sebagai pengingat prosedur standar penerimaan pesanan.
